`&lt;b&gt;` in the HTML SOURCE; when the browser parses the `title`
ATTRIBUTE VALUE, HTML entities in attribute values DO get decoded back
to their literal characters (that's normal attribute-value parsing) —
but since attribute values are NEVER re-parsed as markup, those
decoded literal `<b>`/`</b>` characters just show up as VISIBLE TEXT in
